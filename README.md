# 🎬 AI Stream Cutter

> 🚧 **Проект находится в активной разработке.**

AI Stream Cutter — локальный инструмент для автоматического поиска лучших моментов на длинных стримах, игровых видео и подкастах.

На текущем этапе проект умеет:

- 🎙️ Расшифровывать речь с помощью Faster-Whisper
- 🎧 Анализировать громкость аудио
- ⭐ Оценивать эмоциональность моментов
- 📊 Ранжировать лучшие фрагменты
- ✂️ Автоматически вырезать TOP-N клипов через FFmpeg

Все вычисления выполняются **локально**, без использования облачных сервисов.

# Установка

## 1. Клонируем проект

```bash
git clone https://github.com/YOUR_USERNAME/clip-ai.git

cd clip-ai
```

---

## 2. Создаем виртуальное окружение

### Windows

```powershell
python -m venv venv

venv\Scripts\activate
```

### Ubuntu

```bash
python3 -m venv venv

source venv/bin/activate
```

---

## 3. Устанавливаем зависимости

```bash
pip install --upgrade pip

pip install -r requirements.txt
```

---

## 4. Устанавливаем FFmpeg

### Windows

Скачать:

https://ffmpeg.org/download.html

Добавить FFmpeg в PATH.

Проверить:

```bash
ffmpeg -version
```

---

### Ubuntu

```bash
sudo apt update

sudo apt install ffmpeg
```

Проверить:

```bash
ffmpeg -version
```

# Использование

Поместите видео в папку

```
input/
```

Например

```
input/video.mp4
```

или

```
input/parts/video_001.mp4
```

Запустите программу

```bash
python app.py
```

После запуска программа покажет список найденных видео.

Например

```
1. stream.mp4
2. parts/part_001.mp4
3. parts/part_002.mp4
```

Введите номер нужного видео.

После анализа программа:

- расшифрует видео;
- оценит эмоциональность;
- покажет TOP лучших моментов;
- предложит сохранить TOP-5 клипов.

Все клипы сохраняются в

```
output/clips/
```

# Планы

Следующий крупный этап разработки — интеллектуальная обработка видео.

Планируется использовать **YOLO** для автоматического поиска объектов на кадре.

Это позволит:

- автоматически находить окно веб-камеры;
- определять лицо стримера;
- отслеживать положение человека в кадре;
- автоматически строить вертикальный кадр 9:16;
- создавать Shorts, Reels и TikTok практически без ручного монтажа.

Главная цель проекта — полностью автоматизировать создание коротких роликов из длинных стримов.
